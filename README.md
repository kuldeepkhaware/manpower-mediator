# Manpower Mediator System

A professional Manpower Mediator & Recruitment Management System developed in Core PHP MVC.

## Features

- Role Based Login
- Client Management
- Supplier Management
- Requirement Management
- Candidate Management
- Candidate Matching
- Deployment Management
- Attendance
- Payroll
- Invoice Management
- Reports & Dashboard

## Technology

- LARAVEL
- MySQL
- Bootstrap 5
- JavaScript
- MVC Architecture

## Installation

1. Clone Repository

```bash
git clone https://github.com/kuldeepkhaware/manpower-mediator.git
```

2. Import Database

```
database/manpower_mediator.sql
```

3. Configure

```
config/config.php
```

4. Run

```
http://localhost/manpower-mediator/public
```
